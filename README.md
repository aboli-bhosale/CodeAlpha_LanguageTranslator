# Language Translation Tool

This project is a Language Translation Tool developed as part of the CodeAlpha Internship.

## Description
The Language Translation Tool allows users to translate text from one language to another using an AI-based translation system.  
It is built using Python and Flask with a simple web interface.

## Features
- Translate text between multiple languages
- Simple and user-friendly UI
- Fast and accurate translation
- Web-based application

## Technologies Used
- Python
- Flask
- HTML
- CSS
- deep-translator

## How to Run the Project
1. Install Python on your system
2. Install required libraries:
   py -m pip install flask deep-translator
3. Run the application:
   python app.py
4. Open your browser and go to:
   http://127.0.0.1:5000/
